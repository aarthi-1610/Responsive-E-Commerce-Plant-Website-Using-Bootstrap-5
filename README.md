The Responsive E-Commerce Plant Website is a student-level web application developed using HTML, Bootstrap 5, and JavaScript to provide an online platform for purchasing plants. The system allows users to register, login, browse various plant products, add items to cart, place orders, provide reviews, and request order cancellation or return.

This project demonstrates the implementation of frontend web development concepts, including responsive design, multi-page navigation, user interaction handling, and client-side scripting. It provides hands-on experience in building a complete e-commerce workflow using modern web technologies.
