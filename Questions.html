Bootstrap – Complete Theoretical Notes

1. What is Bootstrap?

Bootstrap is a free and open-source front-end framework used to create responsive and modern websites quickly. It provides ready-made CSS and JavaScript components.

2. Advantages of Bootstrap

  • Fast development
  • Responsive design
  • Easy to learn
  • Cross-browser support
  • Pre-designed components

3. What is Responsive Web Design?

Responsive design means the website automatically adjusts its layout according to screen size like mobile, tablet, laptop, and desktop.

4. Bootstrap Grid System

Bootstrap uses a 12-column grid system to design responsive layouts using rows and columns.

5. Container in Bootstrap

A container is used to wrap content and align it properly.

  • container → Fixed width
  • container-fluid → Full width

6. Row and Column

Row is a horizontal group of columns. Column divides the row vertically to arrange content.

7. CDN

CDN (Content Delivery Network) allows using Bootstrap online without downloading files. It improves loading speed.

8. Bootstrap Components

  • Buttons
  • Forms
  • Cards
  • Navbar
  • Alerts
  • Modals

9. Navbar

A navbar is a navigation bar used to display menus and links at the top of a webpage.

10. Card

A card is a flexible container used to display text, images, and buttons neatly.

11. Modal

A modal is a popup window that appears over the current page to show alerts, messages, or forms.

12. Utility Classes

Utility classes are helper classes for spacing, colors, alignment, and positioning.

Examples: text-center, mt-3, p-2, bg-primary

13. Breakpoints

Breakpoints define layout changes based on screen size.

  • sm – mobile
  • md – tablet
  • lg – laptop
  • xl – desktop

14. Mobile First Design

Mobile-first design means designing the website for mobile screens first, then scaling up.

15. Bootstrap vs CSS

CSSBootstrap
Manual stylingPre-built styling
More codeLess code

16. Latest Bootstrap Version

Bootstrap 5 is the latest stable version.

17. Why Bootstrap is Used?

Bootstrap saves time, provides professional UI, supports responsive design, and makes development faster.
